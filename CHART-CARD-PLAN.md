## 확정 스펙 — "차트 1개 + 판정 4줄"

**화면 (chart.html 별도 페이지, 종목명 클릭으로 진입 — 0픽셀)**
1. 헤더: 종목명 · 코드 · 현재가 · 등락
2. 차트 1개: **120일 종가 + 20일선 + 60일선 + 저항선 1개(120일 전고점) + 지지선 1개** — 끝. 다른 패널 없음.
3. 판정 4줄 (문장 템플릿 ~10개, 전부 계산값 삽입):
   - 📈 추세: 정배열 여부 · 이평선 대비 위치
   - 💰 수급: 외인·기관 연속일·금액 (우리 최강 데이터)
   - 🔍 관찰 포인트: 지지·저항 가격 두 개
   - 🛡 리스크: 20일선 손절 기준가 · 💣 위험 공시 유무
4. 시안: `mockup_chartcard.html` (승인 완료 형태)

**명시적으로 뺀 것 (다시 넣지 말 것, 넣으려면 이 문서 개정 먼저)**
- 별점·진입 신호 강도 → 검증 안 된 신호 포장 금지 (검증 후 승격 원칙)
- RSI·MACD·볼린저·OBV·거래량 패널 → 실전 기여 미검증 + 투머치 인포
- 핸드드로잉 말풍선 주석 → 유지보수 무겁고 반복감의 주범. 선+가격 라벨로 충분
- 종합 판정 박스 → 판정 4줄과 중복

## 데이터 — chart_pack (스키마 추가만)

- 대상: 오늘의 5선 + 스윙 후보 + 종가 스캔 + watchlist ≈ 25~35종목
