## 1. 아키텍처: 현재 → 목표

```
[현재]
GitHub Actions(크론10, 지연 1~2h) ─ collect.py ─→ data.json ─→ Pages 대시보드
                                  └ notify.py ──→ 텔레그램(텍스트만)

[목표 = 두뇌·심장·감각·인터페이스 4축 보강]
① 두뇌: LLM 분석층 ──→ ai_insights.json (공시번역·테마해설·시황3줄)
② 심장: VPS 정시 크론 (지연 0분) ──→ GitHub에 커밋 (저장소·Pages는 그대로)
③ 감각: KIS 오픈API (실시간 시세·분봉) + KRX 공매도
④ 인터페이스: 텔레그램 차트 이미지 + 장중 목표가/손절 파수꾼
```

핵심 설계 사상: **GitHub(저장소·Pages·이력)는 영원히 무료 기반으로 유지**하고,
유료 요소는 전부 "갈아끼울 수 있는 부품"으로 바깥에 붙인다. 어느 부품을 빼도 현행 무료 시스템으로 복귀.

## 2. Tier 1 · 두뇌 이식 — AI 분석 자동화 (월 500원~5천원) ★가성비 1위

지금 규칙 기반이 못 하는 것 = 문맥 이해. 이걸 LLM이 매일 자동으로.

| 코드 | 기능 | 내용 | 대체하는 것 |
|---|---|---|---|
| AI-1 | 공시 번역기+ | 매일 공시 60건 → 쉬운 말 1줄 + 호악재 -10~+10 + 근거 | IDEAS S7 (규칙 기반 한계 돌파) |
| AI-2 | 테마 해설가 | 뉴스 나침반의 "점화 테마"에 **왜 오르는지** 한 줄 해설 + 수혜 사슬 | 뉴스 나침반 고도화 |
| AI-3 | 시황 3줄 | 아침 브리핑 맨 위 "어제 미국장→오늘 한국장" AI 요약 | 수동 복붙 작업 |
| AI-4 | 주간 복기 코치 | 일요일: 내 5선·스윙 성적 + 시장 맥락 → 튜닝 제안 서술형 | 주간 결산 고도화 |

**플로우**: Actions(or VPS) 수집 완료 → 뉴스타이틀·공시목록을 LLM API에 배치 1회 호출
(JSON 모드 강제) → `ai_insights.json` 커밋 → 대시보드 카드 + 텔레그램 라인.
LLM 실패/한도 초과 시 → 기존 규칙 기반 문구로 자동 폴백 (조용한 실패 없음, 테스트 픽스처 포함).

**비용 산출** (하루 입력 ~10만 토큰: 뉴스 500건+공시 60건+지시문, 출력 ~1만 토큰, 월 22영업일):

| 모델 | 단가(입력/출력, 1M토큰) | 월 예상 | 특징 |
|---|---|---|---|
| Gemini 2.5 Flash-Lite | $0.10 / $0.40 | **약 500원** | 최저가, 품질 무난 |
| Gemini 2.5 Flash | $0.30 / $2.50 | 약 1,500원 | 균형 |
| Claude Haiku 4.5 | $1.00 / $5.00 | 약 4,500원 | 한국어 뉘앙스·판정 품질 ↑ |

→ **파일럿은 Flash-Lite(월 500원)로 시작**, 판정 품질 불만이면 Haiku로 승격.
AI-4까지 다 켜도 월 5천원 안쪽. 예산이 아니라 "판정을 믿을 수 있냐"가 진짜 관문이므로,
첫 2주는 **AI 판정 vs 기존 규칙 판정을 나란히 표시**해 검증 후 정식 채택.
